## 🎯 内容策略调整

### 核心原则
- **避免产品词**: 不提"花束"、"鲜花"等具体产品
- **聚焦场景**: 强调"送礼"、"表达心意"、"祝福"、"慰问"
- **情感连接**: 围绕"心意"、"祝福"、"关心"展开

### 文案方向示例
| 场景 | 关键词组合 | 文案方向 |
|------|-----------|----------|
| 送妈妈 | 送妈妈 + 表达心意 | "妈妈，我想你了，送您一份小心意" |
| 妇女节 | 妇女节 + 送女性 + 表达祝福 | "38女王节，给她的祝福" |
| 生日 | 生日礼物 + 表达心意 | "生日惊喜，让TA感受到你的心意" |
| 清明 | 清明节 + 表达慰问 | "清明祭扫，表达怀念与慰问" |
| 日常 | 微信送礼 + 线上送礼 | "微信送礼新方式，心意直达" |

## 📊 性能监控指标

- **关键词覆盖率**: 实际命中的关键词数量 / 总关键词数
- **互动率**: 评论点赞数 / 帖子曝光量
- **转化率**: 微信搜索"卡美心礼"次数 / 总互动数
- **平台分布**: 各平台产生的内容数量占比

## 🔧 手动更新流程

如需手动更新关键词：

1. 编辑文件: `marketing/social_media_cron_config.json`
2. 修改 `payload.message` 中的关键词列表
3. 更新 `payload.thinking` 中的产品文案策略
4. 测试任务执行: `openclaw cron run <job-id>`
5. 验证输出文件是否包含新关键词
