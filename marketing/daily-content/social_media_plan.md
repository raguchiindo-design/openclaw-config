# 社交媒体智能互动系统 - 详细规划

## 🎯 系统概述
基于cron jobs、WebMCP和agent-browser的社交媒体智能互动系统，专注于全民k歌、美篇、Soul、微博平台，针对特定关键词生成互动内容。

## 📊 平台与关键词

### 🌐 目标平台
1. 全民k歌
2. 美篇
3. Soul
4. 微博

### 🔑 核心关键词
- 怀旧，老歌，思念
- 送礼，送领导，送妈妈，送客户
- 2026年马年祝福，年货，马年，除夕
- 情人节，致歉，送上级

## 🤖 任务流程设计

### 每小时任务（30分钟间隔）
1. **平台扫描**：使用agent-browser访问各平台
2. **WebMCP分析**：分析页面逻辑和热门内容
3. **关键词匹配**：识别相关帖子
4. **内容生成**：生成10个帖子标题和链接
5. **评论建议**：生成亲切/活泼版本评论
6. **数据保存**：保存到marketing目录
7. **telebot通知**：完成任务通知

### 📝 内容策略

#### 不同人群文案
- **给父母**：使用"金祥云兰+定制祝福"，强调亲情和祝福
- **给闺蜜**：使用"超光·飞骏"，强调友情和心意  
- **给父辈**：使用"2026火马年-赤兔红马🐎"，强调传统和尊重
- **给美女**：使用"超光·飞骏"，强调优雅和赞美

#### 评论建议要求
- 亲切版/活泼版
- 像朋友分享经验，先共情再说经历
- 可轻提「祥云花店」
- 提示在微信搜索"卡美心礼"
- ≤50字

## 📋 待办清单格式
```
帖子简述+链接+适合原因+评论版本
```

## 📅 Cron Job 配置

### 基础配置
```json
{
  "name": "社交媒体智能互动",
  "schedule": {
    "kind": "every",
    "everyMs": 1800000, // 每30分钟
    "anchorMs": 0
  },
  "payload": {
    "kind": "agentTurn",
    "message": "执行社交媒体智能互动任务：扫描平台，生成互动内容",
    "model": "glm/glm-4.6v",
    "thinking": "使用WebMCP分析社交媒体页面逻辑，生成符合关键词的互动内容"
  },
  "sessionTarget": "isolated",
  "enabled": true
}
```
